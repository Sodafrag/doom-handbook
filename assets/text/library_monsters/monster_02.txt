### **幼魔（Imp）**

> “你以为它们是童谣里拿着干草叉穿着红色连身衣只会恶作剧的顽皮小鬼，然而这些地狱大军马前卒实际上即残暴又敏捷。低阶恶魔们在每一个地狱势力踏足的战场上狂欢，贪婪地吞噬着战败者的血肉。”

Imp这个名字来源于日耳曼民间传说中所描述的一种虚构生物。词源通常认为来诞生“ympe”，在时代的演变中可以是小恶魔、女巫甚至是用来代指一棵嫁接的树。如今它一般被视为某种活泼调皮的小怪物的专属名称。

通常来说，Imp是一种个头小巧，充满野性且喜爱恶作剧的小魔精。与基督教神话中纯粹邪恶的魔鬼形象不同，日耳曼神话里的恶魔立场通常是相对模糊的。虽然经常会做出一些让人头疼的举动，但有时候也会以神仆的身份帮助人类。

某种程度上来说，id将它们在游戏里设计成了视杀戮为嬉戏的习性，也算是一种对恶趣味式的讽刺了。

幼魔是一种与人类体格相近的类人怪物。它们在系列所有作品里都始终作为最常见的低阶恶魔杂兵活跃在整个流程中。可以说，不论是哪部作品里，构成整个关卡最底层结构的杂兵非幼魔莫属。

它们在所有作品中的行动基本上都不会脱离灵活轻盈，通过远距离投掷火球和近距离爪子撕扯的战斗方式。虽然单个幼魔并不能对主角造成什么太大威胁，但不论是集群狩猎强大的对手，还是牺牲个体满满消耗你的体力都让它们成为团队中不可或缺的成员。

在最初的设计中，幼魔原本是以参考电影《异形》里的外星生物为基础。虽然在初代中并没有展现出与其太多相同的特征，但后来从《毁灭战士3》到后续所有作品里，倒可以更加明显地看出其对异形的致敬与参考。

另外，旧系列的各种民间MOD里幼魔也经常被修改替换成异形，也算是玩家自发对灵感源头的考据吧。



- **元祖幼魔**

*——相貌容易让人产生厌恶的褐色低阶恶魔，在杀戮时总能保持“微笑”*

| 属性       |                     |
| ---------- | ------------------- |
| 攻击模式： | 弹丸抛射/ 近战      |
| 伤害：     | 3-24 / 3-24（快速） |
| 生命值：   | 60                  |
| 硬直概率： | 79.30%              |

![](resource:assets/images/library_monsters/m02_01.jpg)

这些初级恶魔只是些速度平均，靠着偷袭得手的家伙。它们有着褐色的皮肤、眼睛与嘴巴冒着红光，身上还有这重金属风格的骨刺。它们在停机坪就能被目击到，是地狱中数量最多但也是最低贱的恶魔阶层。



> **设定资料与战术建议：**
>
> “元祖”幼魔并没有它后续版本里灵活矫健的特性，在游戏里的经典造型在之后系列最新作的《毁灭战士：永恒》里得到了还原。
>
> 幼魔在待机状态下会时不时发出吞咽的咕噜声，玩家可以通过这个来判断附近是否有埋伏。它们在发现玩家后会有一声嘶吼以作为“警告”，然后以远距离投掷火球的方式进行偷袭。通常情况下，一个在你视线内发射的火球可以轻易回避，但并非所有情况都那么顺利。
>
> 首先自然是它们有很多会在远距离就进行攻击，如果你正好在其弹道上与其它敌人作战，很可能就会莫名其妙吃上几发火球。而更多时候，它们经常会被关卡设计师放在危险、狭窄的地形里，形成令人恶心的火力网。
>
> 用手枪对付它们虽然可行，但还是用霰弹枪收效更好。如果对方数量较多，那么拿出火神炮显然是最合理的选择。不推荐使用近战，因为其相对较高的攻击频率，你很有可能在靠近后就挨上一发火球。
>
> 幼魔的火球仅对同类有着免伤效果。基于这个因素加上其战斗特征，如果敌人是混编制的，幼魔便成了你最不需要优先处理的目标。你大可以养成一种将其留在最后解决的思考模式。因为在此之前，它们多数情况下已经因为被僵尸士兵击中、或是误伤了跑在前头的粉红魔而引发一场内斗。这些倒霉蛋几乎从来都不可能在狗咬狗的情况下获胜。
>
> 《毁灭战士64》里得到了新造型的幼魔虽然数值没有变化，但攻击频率变得更高。并且还增加了一种名为“梦魇幼魔”的亚种，其特征是肤色带有难以辨识的迷彩效果。
>
> 豆知识：在卡马克将源代码免费公开后，人们通过其中的数据挖掘推断出，幼魔最初形象可能更加接近传统民间神话里的造型。
>



- **DOOM3幼魔**

*——和远房亲戚异形不同，它们认为眼睛越多越好*

![](resource:assets/images/library_monsters/m02_02.jpg)

它们与20世纪某个叫做《异形》的老电影出现的怪物十分相似，不仅会发出尖锐刺耳的叫声，还喜欢在通风管道和阴暗地段里偷袭。它们可以进入狭窄的地形，或是在天花板上潜伏，对你进行前后夹击。它们投掷火球需要进行一次蓄能。研究资料显示这是一种来源不明的等离子体能量。通过10只眼睛，它们能够在阴暗的环境下辨识物体。

在进入三角洲实验室后，陆战队员发现彼苏格博士的团队其实早就已经捕获了一些样本。从研究资料里显示，它们发达的四肢无论是双足行走还是四肢着地爬行都十分敏捷，并且可以利用强壮的爪子在垂直平面上攀爬。

奇怪的是，在进入地狱后你遭遇到的幼魔形象有了改变。它们似乎不再拥有黑色的坚硬表皮，看起来像是被剥掉了一层皮般的血肉质感。这可能证明了幼魔拥有非常惊人的快速适应力，在前往火星后就改变了自己的生理结构以融入环境。



> **设定资料与战术建议：**
>
> 在拥有3D图形建模后，这些生物得到了更加灵活和具有非人类习性的特征。使用霰弹枪对付它们是很好的选择。因为其较笨拙的AI使得你在近距离爆头也没什么难度。不过，《毁灭战士3》里的地形狭窄，角色机动能力与近战能力堪称系列最弱，如果被这些幼魔堵在墙角那可不好受。
>
> 因为火球飞行速度较慢，你仍然可以比较轻松地回避攻击。有时候，它们会躲在一个必经之路的门后面，你一旦靠近便会遭到飞扑袭击。不过老实说，这种攻击由于伤害很低，所以除噩梦难度之外只能算是吓唬你的一直恶劣手段罢了。
>



- **2016幼魔**

*——地狱里随处可见的弱者；通常被其它恶魔视为小弟…或蝼蚁*

![](resource:assets/images/library_monsters/m02_03.jpg)

幼魔的身手敏捷，几乎没有任何地形可以拦住它们的行动。尽管在地狱中处于弱者的位置，但它们仍然能够使用某种“魔法”——通过在手中转化凝聚地狱能量，以制造出一个可以进行投掷的高温火球。

这些生物暗示了恶魔可能存在“进化”的生态现象。从文档资料中得知UAC曾经对捕获的幼魔进行解剖对比，发现它们很可能是恶魔召唤师的低阶品种。此外，一些早期目击影像里出现了与如今造型有一定差异的幼魔——很容易就让人联想到与地狱骑士之间的亲戚关系。



> **设定资料与战术建议：**
>
> 幼魔是在《毁灭战士2016》里得到最多强化的经典敌人之一。它们比过去更快更灵活，有着准确高效的攻击手段。作为地狱军团最低阶的士兵，它们成群结队地涌入几乎每一场战斗，很好地完成了从炮灰、骚扰者到炮手的任务。
>
> 就像游戏中文档资料所述的那样，幼魔在战斗中会采取多种多样的手段，以期能够填补整个团队中的每一道缝隙。它们会在远处对玩家进行火球投掷，这其中也分为不断进行机动骚扰、准头较差的攻击；也有在安全距离下站成一排不断进行火力压制的情况。
>
> 另外，相对较为罕见的情况是它们“抽到”了发动强力攻击的指令。这其中有悬挂在支柱上充当狙击手的特殊状态，也有进行原地蓄能，丢出一颗大型火球的“必杀技”。
>
> 在高难度下，幼魔比看上去的要危险许多。它们高速且准确的攻击效率会让你在原地站立3秒之内便遭到击杀。一次准确的火球在噩梦难度下可以造成45点伤害，而蓄能版本更是有着到惊人的75点。尽管它们很少主动进行贴身战斗，但一旦与你离得够近便会用爪子进行连续打击。
>
> 不建议在它们身上浪费太多时间与资源，但也不应当轻视这些长期对手。使用突击步枪来对付它们是最好的选择。利用这把武器精准快速的优势将它们的脑袋一个个打爆，然后专心去面对更强的对手。另外像是战斗霰弹枪的榴弹模组、或是火神炮与等离子步枪也能有效地收割这些烦人的家伙。
>



- **DOOM永恒幼魔**

*——神话寓言中在地狱焦土上狂奔的野兽之一，如今在显示中享用着无辜者的灵魂*

![](resource:assets/images/library_monsters/m02_04.jpg)

幼魔原本是地下世界的拾荒者与清道夫，只能苟延残喘在高等恶魔的阴影之下，偶尔捡一些残羹剩饭。这与它们的力量相符，比较作为地狱中的低阶恶魔它们的力量极其有限，并且仅仅受到其狩猎本能驱使。

然而，这种天生的残暴也令其十分适合作为一种完美的军队士兵存在。地狱的至高王权将其视为一种易于操控、数量庞大的炮灰，部署在各个军团方阵当中。你会看到它们被驱赶在进攻的第一线，用无情且对敌我双方都十分残酷的自杀性战斗方式冲垮对方防线。

> 
>
> **设定资料与战术建议：**
>
> 《DOOM：永恒》里的幼魔和恶灵士兵一样，都是在造型上更加还原经典质感，但战斗力与前作无异的炮灰。同样在玩家已经变得更加强大的情况下，它们已经不是什么能被视为威胁的存在了。
