### **疼痛元素（Pain elemental）**

> “俗话说死掉的恶魔才是好的恶魔，但唯独痛苦元素死了和活着一样很糟糕。”

痛苦元素是与巨脑魔不同的另一种空中火力单位。它们的性质更接近于活体航空母舰，能够无限制的喷射出迷失丧魂。如果放任不管，将会导致极为混乱的情况出现。



#### **元祖痛苦元素**

*——觉得你的弹药储备足够？当你遇到这些空中肉丸子时就不会那么认为了。*

| 属性       |          |
| ---------- | -------- |
| 攻击模式： | 敌兵生产 |
| 伤害：     | 自身无   |
| 生命值：   | 400      |
| 硬直概率： | 50%      |

![](resource:assets/images/library_monsters/m11_01.jpg)

一种长相与巨脑魔十分相似的怪物，两者之间应该是有什么生态上的联系。它们都是漂浮在空中的球型恶魔，长着独眼与血盆大口。不过痛苦元素的皮肤是褐色的，并且还有两只与身体比例不符的短手臂。

从技术上来说，真正与痛苦元素有紧密联系的其实是迷失亡魂。如果不是能够随心所欲从嘴里“吐”出这些尖叫的飞行骷髅的话，痛苦元素在地狱军团中恐怕也只能作为肉盾...或是便携军粮。



> **设定资料与战术建议：**
>
> 痛苦元素没有直接的战斗力。但它们的AI逻辑仍然会在瞄准玩家后进行一次“发射”动作。这时就会从痛苦元素的嘴里生成一只向前冲撞的迷失亡魂，并在被消灭之前作为新敌人永久停留在关卡中——这是原始游戏中除BOSS外唯一能“添加”敌人的情况。
>
> 虽然当玩家隐蔽在它们视线之外的时候，痛苦元素通常不会主动进行攻击，但是它们却会因为被误伤而向其它恶魔吐出迷失亡魂。而且往往误伤它们的正是自己吐出来的小弟......
>
> 由于这种性质，痛苦元素变成了最不应当被触发怪物内斗的敌人。玩家一定要在事态变得不可收拾之前击杀痛苦元素。
>
> 痛苦元素在死亡时会爆炸变成最多三只迷失亡魂，因此也不会留下尸体。然而某些情况下——诸如被压扁而导致变成烂肉时，它们可以被恶拱所复活。此时游戏会出现BUG，导致被复活的痛苦元素变成无法被攻击到的“幽灵敌人”。
>



#### **DOOM永恒痛苦元素**

*——只有通过杀戮，它们才能从无尽痛苦中获得一丝喘息机会*

![](resource:assets/images/library_monsters/m11_02.jpg)

在暗影平原最古老、也是最令人憎恶的深渊中，痛苦元素从中诞生而出。它们受到了无尽的诅咒，被用作制造迷失丧魂的媒介——从它们如同火坑一般的肚子里。

这种共生关系让痛苦元素如其名字一般，终其一生都在无休止地遭受折磨。唯一能让它们在这苦难命运中获得一丝宽慰与满足的方式，就是将体内的迷失丧魂发射出去，并看着它们将受害者残忍杀害的场面。这也是痛苦元素名字的另一面：给他人制造苦痛。



> **设定资料与战术建议：**
>
> 造型比过去更加狰狞的痛苦元素在缺席了两部作品后得以回归。如今，这些获得了生命值与护甲增强的航空母舰是后期十分难缠的敌人，它们会在远距离投掷迷失丧魂，对玩家造成相当可观的伤害。
>
> 和过去制造混乱的设计不同，《DOOM：永恒》的痛苦元素更像是巨脑魔进阶型态。它们吐出了迷失丧魂后会主动将对玩家进行投掷，也会让其围绕在自己身边作为护盾。整体来说，痛苦元素没有特别需要注意的设计，是一个纯粹攻高血厚难以处理的高阶空中肉盾。
