### 1.1游戏基础版本

那些多数诞生于90年代，由id Tech 1引擎开发的作品，一般都统称为“元祖毁灭战士”。它们作品版本繁多，并且可能拥有游戏史上登陆平台数量最多的移植版本。更何况提不同主机平台上的内容也各有差异。因此要全部记录并展示在本书中即麻烦，也没有太大必要——毕竟我想也没多少人在乎一些古董计算机系统的DooM。此外，这里并不会记录那些诸如在提款机、吸尘器、超市LED显示屏或电子手表上运行的非官方民间版本。统计元祖毁灭战士仍旧不是什么轻而易举的事情，因此这里仅列出最常见的几个版本。

### **毁灭战士（DooM）**

| 开发团队：id Software                                        | 初版发售日：1993年12月10日                                   |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 创意总监：汤姆.霍尔                                          | 设计师：约翰.罗梅罗、桑迪.彼得森、汤姆.霍尔                  |
| 程序员：约翰.卡马克、约翰.罗梅罗、戴夫.泰勒                  | 编剧：汤姆.霍尔                                              |
| 美术设计：艾德里安.卡马克、凯文.克劳德                       | 作曲：罗伯特.普林斯                                          |
| 平台：DOS系统、Windows、Jaguar、Linux、PC-98、SEGA 32X、SNES、3DO、GameBoyAdvance、Xbox 360、iOS、Nintendo Switch、PlayStation 4，Xbox One、Android | 发行商：GT Interactive、ActiVision、Bethesda Softworks、Sega、Atari Corporation、Williams Entertainment、Ocean Software、Imagineer |

最初的《毁灭战士》采用第一章免费向市场与互联网传播、后续版本付费订购的形式进行发售。到1995年，《毁灭战士》已经达到了超过1000万台计算机的用户群体。尽管大多数用户未购买注册版本，但它仍然是当时最赚钱的游戏之一。通过这种形式，id公司可以回避发行商和版税而获得80%以上的净利润。

更重要的是在此之上，整个西方亚文化彻底因为DooM而“出圈”——有点类似2020年《集合吧！动物森友会》的情况。不仅仅是整个玩家群体都成为了游戏的粉丝，就连从来没接触过这个领域的人也不得不对其表示关注。

游戏讲述了孤独的无名陆战队员，自身抵抗火星军事基地实验事故导致的恶魔入侵事件。作品的主要灵感来自于id公司成员某次《龙与地下城》的跑团（团灭）经历。而在项目确立后，他们也对其融入了个人喜爱的8、90年代流行文化元素。其中以《异形》、《鬼玩人2》和克苏鲁神话为主，另外每个关卡中也暗藏了不少摇滚乐或是其它电影的彩蛋。

《毁灭战士》并非首个第一人称动作射击游戏，但它其中所开创的诸多元素和规则，彻底巩固了这个类型的基本概念，许多设计至今都还在影响着后续者。玩法以强调解谜、探索以及高速战斗为核心，其千变万化的关卡设计使其在简单的规则下仍然能保持丰富变化。此外，游戏开创了死亡竞赛和战役合作模式，让游戏厂商看到了如何有效地延续一款作品后续几年生命力的做法。

在发售了一段时间后，游戏程序员约翰.卡马克大力推行了向民间传播免费的玩家自创模组（MOD）文化，甚至还免费公开了所有相关技术的源代码。这个行为让所有人都可以加入到对游戏内容的创作与修改上，使得这款距离现在将近30年的老游戏至今都有着惊人的活跃性。更重要的是，它也是如今大量游戏从业人员投身其中的关键契机。可以说在此之后的欧美游戏行业，有无数人受惠于id的贡献与成就。

和如今你能在各个数字游戏销售平台购买到的版本不同，事实上我们现在根本买不到真正的《毁灭战士》。这个版本实际上是Id最初在1993年以完全免费的共享软件形式向市场投放的。和多数现在的玩家所接触到的、总共有4大章节的《终极毁灭战士》不同，原始版本只有三个章节。也因为如此，我们才会在第三章就看到结局部份打上了“剧终”字幕。

当然了，如果你愿意花800美元级别的起底价，去一些古董竞标网站上寻找的话，大概还是能买到《毁灭战士》原始版本的。而且这东西的收藏价值仍旧在不断飙升…

### **终极毁灭战士（The Ultimate DOOM）**

- 发售日：1995年4月30日

在看到了零售版本的《毁灭战士II：人间地狱》的成功后，原本不打算与他人合作的id开始对商业发行模式兴趣大增。它除了采用新包装和版号发售之外，还在原始版本上做了优化与增强。为了公平对待之前已经购买了原始版本的玩家，id免费为他们提供了升级补丁。

在该版本上市之前，id原本的计划是采用《至高威能毁灭战士特供定制版》来命名。很可惜因为大家都觉得太过于愚蠢浮夸而被驳回。但这个名字仍然可以通过源代码和游戏DOS安装文件夹里找到尚未修改的程序名称。

该版本的新增内容——《噬尽汝肉》由约翰.罗梅罗主导设计。此外后来加入的第二批id成员如蒂姆.威利茨和亚美利坚.麦基，以及两位从民间征集来的粉丝也参与了许多其中几个关卡的创作。id的美术小组为该章节做了一些新的图形素材，但基本上游戏没有实质性的新增内容。

除了新的关卡之外，罗梅罗还对最初的E1M1关卡进行了重新设计。所以如今我们玩到的版本，其实并非完全纯粹体验的原始内容。

由于在原始版本的结局已经衔接了《毁灭战士II：人间地狱》的开场，所以这个新章节强行增加的故事一直处于争议中。无论是发生的时间还是地点，都与剧情产生了一定程度的冲突，所以通常来说无论是官方还是粉丝都对其进行了模糊处理。一般只认为是主角在原始版本结束后，遇到了敌人垂死挣扎的抵抗，最后在一番作战后再接上了《毁灭战士II：人间地狱》的开场部份。

到了今天，如果你在各大销售平台上购买所谓的初代《毁灭战士》，只能得到这个强化版本。因为像上面说的那样：原本只有三个章节的元祖实体版，如今已是千金难求的收藏品了。

### **Doom95**

- 发售日：1996年8月20日

尽管在1995年左右，微软公司为Windows 95操作系统花费了数百万美金，但比尔盖茨还是沮丧地发现：它的装机量始终被《毁灭战士》压在底下。作为当时几乎所有北美个人计算机上必定安装的程序，Doom的影响力不仅远远超过了当时史上已知任何一个电子游戏，甚至还让微软相形见拙。

为此，比尔盖茨制订了一套靠Doom来推广Windows 95的营销策略。其中之一就是由加布.纽维尔（Gabe Newell——G胖）领导的团队负责开发、号称专为所有Windows用户准备《Doom 95》。 这个专属端口支持四种屏幕分辨率，有一些新的功能与相比DOS系统更加便利的设置方式。微软希望以Doom来终结MS-DOS系统的时代——别笑，人家当年真这么对外宣布的。

然而事实上，这个版本在关卡内容上有一些删减，并且带来了诸多Bug。此外它也无法播放MIDI音乐，而是采用了MUS。相比之下在声音上没那么“经典”。对于在这一细节颇为看重的Doom玩家来说是个很难忽略的问题。

有趣的是，如今你在国内的一些盗版游戏下载网站上，居然找到的多数都是这个版本。

### **印记（SIGIL）**

- 发售日：2019年5月26日

在初代游戏诞生将近26年后，由约翰.罗梅罗制作的全新章节“印记”（SIGIL）发布。该章节以实体盒装收藏版预订的形式发售，售价166美元。该收藏版包括了一个软盘造型的U盘，此外比如说明书、艺术设定集和海报、硬币和罗梅罗本人雕像、T恤等等收藏品。当然还包括这位明星制作人亲笔签名的包装盒。

而在实体版出货后不久的5月31号，数字版也已免费下载的形式上架。另外，贝塞斯达于2019年在新平台上发售的移植版，后来也免费更新了这个章节的下载。至此，在初代发售时隔二十多年后，《毁灭战士》全部5个章节终于全部得以完成。

### **毁灭战士2：人间地狱（Doom II: Hell on Earth）**

| 开发团队：id Software                                        | 初版发售日：1994年10月10日                                   |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 创意总监：桑迪.彼得森                                        | 设计师：桑迪.彼得森 / 肖恩.格林 / 亚美利坚.麦基              |
| 程序员：约翰.卡马克 / 约翰.罗梅罗 / 戴夫.泰勒                | 编剧：不明                                                   |
| 美术设计：艾德里安.卡马克 / 凯文.克劳德                      | 作曲：罗伯特.普林斯                                          |
| 平台：DOS系统、Mac OS、PlayStation、Sega Saturn、Game Boy Advance、Tapwave Zodiac、Xbox、Xbox 360、PlayStation 3、PlayStation 4、Nintendo Switch、Xbox One、Android、iOS | 发行商：GT Interactive、Virgin Interactive、Entertainment、Activision、Bethesda Softworks |

《毁灭战士II：人间地狱》是id第一款正式委托发行商代理，以商业软件形式销售的DOOM系列作品。玩家这次将回到地球，凭借一己之力（和无数弹药）对抗入侵家园的地狱军团。

游戏的玩法与前作相比几乎没有任何改变。在内容上，它增加了几种新的敌人、添加了一把名为[超级霰弹枪]的新武器。而在流程机制上面，本作一改可选择的章节形式，将全部32个关卡放到一整局游戏里。这个流程形式最直观的改变，就是只要没有在中途死亡，那么获得的武器资源可以一直保留到通关。

由于前作共享软件版本打下的基础，游戏成为了1994年～1999年最畅销的计算机游戏之一。这个形式也改变了id对商业市场的看法，从此逐渐转型为与同行一样的正规商业工作室。

尽管游戏在已经出现许多竞争对手的市场上仍然一帆风顺，但相比前作在粉丝群体里的口碑却明显下滑。尤其是过于强调数量的敌人配置，以及庞大却不如前作精巧的关卡设计都令人有些失望。在游戏发售多年后，不少从当时走过来的系列玩家都认为其过于微小的改变不符合“续作”的标准。

### **大师级毁灭战士**2（Master Levels for DOOM II）

- 发售日：1995年12月26日

在《毁灭战士》发售了一段时间后，id发现市面上有人利用他们对版权的宽容态度中饱私囊。一家过去专门生产低成本劣质游戏的公司——WizardWorks推出了一个名为《D!ZONE》的软件系列。这东西利用《毁灭战士》便利的关卡设计功能，推出了各种粗制滥造的地图包。

虽然id允许人们随意使用游戏的引擎技术与模组开发，但在这件事情上，WizardWorks这种以公司身份拿别人东西贩卖的行为明显属于钻空子侵犯权利。

id并不打算付诸法律途径解决，毕竟他们也不在乎对方这点蝇头小利。但为了反击这种厚脸皮的行为，他们决定推出自己的民间玩家模组合集。于是在1995年底，以《毁灭战士II：人间地狱》扩展包形式推出的《大师级毁灭战士》上市了。它以CD光盘的形式出售，其中包含了二十个WAD文件。这些地图包全部是在id监制下，由社区多位玩家根据合同要求创作。其中名为TEETH.WAD的文件除了正式内容外还包含一个隐藏关，因此这个扩展包实际有着21个关卡。

最后值得一提的是，id还在这个扩展包里玩了一些互联网花样。玩家可以通过预购实体版获得奖励代码，在网上下载海报和订购主题T恤等等。

### **32-bit平台版本毁灭战士**

- 发售日：1995年11月16日

虽然《毁灭战士》的流行让它在90年代初被移植到许多平台上，但硬件方面的限制了家用机版本无法还原其完美品质。直到1995年底，id利用PlayStation的3D硬件功能对引擎做了优化后移植的版本，才让玩家得以在家用游戏机平台上体验到较为完整的《毁灭战士》。

这个版本的渲染效果得以加强，比计算机上呈现出了更加鲜艳多样的色彩。而在流程方面，它分别挑选了初代的28个、续作的23个以及7个原创关卡合并到一起，最终使得本作的单局长度可谓达到了整个系列之最。另外游戏还可以用两台主机连接后的多人游戏功能，但并不支持PS1时代常见的单台主机分屏游玩。

约翰.罗梅罗再次吹破牛皮。他对该版本的评价是“迄今为止最好的DOOM”，而这句话也被印在了游戏包装盒上。然而实际呈现的效果并没有超越计算机版本，最后罗梅罗不得不在访谈被询问时支支吾吾说道：*“我并没有将计算机上更好的操作方式算进去...”*

PS版本的一个贡献是它优化了引擎和端口，使得后续作品得以更容易制作。之后诸如世嘉土星游戏机的移植，甚至脱胎换骨的《毁灭战士64》都是基于该版本的基础而来的。

### **最后的毁灭战士（Final Doom）**

| 开发者：TeamTNT / Casali brothers / id | 发售日：1996年6月17日                    |
| -------------------------------------- | ---------------------------------------- |
| TNT：Evilution制作人：泰.哈德曼        | Plutonia Experiment制作人：达里奥.卡萨利 |

《最后的毁灭战士》是以《毁灭战士II：人间地狱》为基础的修改版本。它的游戏玩法完全没有任何改变，仅在引擎上做了优化，并加入一些新的图形素材与配乐。其内容包括了两个由民间玩具创作的WAD——《TNT：Evilution》与《Plutonia Experiment》。

该版本诞生的契机十分“功利”。在看到了《大师级毁灭战士》仍然能够在市场上吃香后，约翰.罗梅罗认为这种征集社区玩家作品贩卖的形式依然有利可图。于是id继续在民间物色自己看中的玩家创作，打算推出体量更大的扩展版本。

一个名为“TeamTNT”的粉丝团队创作了《TNT：Evilution》。这个质量上乘的关卡MOD原本是要以免费共享的形式发布在互联网上面的。然而就在即将上传的前几天，id突然宣布将会买下这个WAD，以商业销售的形式发布。

然而这个行为却带来了争议。它可以说是id第一次真正意义上遭到负面评价的起因。玩家对这个行为感到非常愤怒，并以无视其品质，仅凭其破坏信誉的代价将《TNT：Evilution》列入“十大臭名昭著的WAD”榜单中。

而《Plutonia Experiment》是以社区玩家提供的作品，由id内部成员修改优化而来的。它的关卡基本上都是一些没能完成，带有实验与练习性质的作品。id在筛选了这些“投稿”后，由罗梅罗和亚美利坚.麦基带头进行管理和监制，最后挑选出了能够使用于正式游戏中的一系列关卡整合而成。由于各种各样的因素，《Plutonia Experiment》的难度之高令人叹为观止。如果不利用作弊行为、并选择能够发挥100%内容的[超暴力！]难度的话，它可以毫无悬念地当选游戏史上最难的挑战之一。

2020年初，已经收购了id的发行商贝塞斯达在新端口版本的原始DooM游戏上更新了一个补丁。该补丁除了修复一系列问题之外，也追加了官方推送的Wad内容。其中第一批作品就包含了《SIGLI》和《Final Doom》。

### **毁灭战士64（DooM 64）**

| 开发团队：Midway Games                                       | 初版发售日：1997年4月4日                              |
| ------------------------------------------------------------ | ----------------------------------------------------- |
| 监督指导：id Software                                        | 设计师：兰迪.埃斯特雷拉 \ 蒂莫西.海德拉 \ 丹尼.刘易斯 |
| 程序员：亚伦.塞勒                                            | 编剧：不明                                            |
| 美术设计：苏克鲁.吉尔曼                                      | 作曲：奥布里.霍奇斯                                   |
| 平台：Nintendo 64、Windows、Nintendo Switch、PlayStation 4、Xbox One | 发行商：Midway Games、Bethesda Softworks              |

由于无法抽出更多时间，id将原本于1994年立项、名为《赦免》的DOOM相关项目交托给了《真人快打》开发商Midway。最终，专门为任天堂64主机准备、由两家公司合作开发的《毁灭战士64》于1997年初上市。

本作的玩法理念与过去一样：探索迷宫、获取物资、并在与群魔的战斗中生存下来。但是本作的图形、渲染技术和数值等等都有了明显的改变，相比几乎毫无变化的《毁灭战士II：人间地狱》来说它反而更被粉丝推崇为“真正的续作”。

游戏的剧情被视为旧系列故事的终结。玩家扮演的无名陆战队员在回到疑似火星基地后，击败了被称为“邪恶老娘”的BOSS，最后独自留在地狱以保证恶魔们无暇入侵地球。而这段剧情由于因在多年后的《毁灭战士2016》中，出现了多处互相呼应的暗示，被经历过旧系列的老玩家，视为新作与旧系列之间的联系。
