### **DOOM（2016）**

| 开发团队：id Software（主要）Panic Button（NS移植）Certain Affinity & Escalation Studios（协助） | 发售日期：2016年5月13日（PC，PlayStation 4，Xbox One） / 2017年11月10日（任天堂Switch） |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 创意总监：马蒂.斯特拉顿，雨果.马丁                           | 引擎 / 程序员：id Tech 6 / 比利.伊桑·汗，提亚戈.索萨，约翰.卡马克（鸣谢） |
| 编剧：亚当.加斯科因                                          | 作曲 / 演奏：米克.戈登                                       |

早在2008年左右，id就确认了将会开发名为《毁灭战士4》的系列最新作。然而在接下来很长一段时间里，这款游戏都如同深藏在迷雾当中；无论是具体的游戏内容、开发进度还是上市日期都没有任何信息透露。

直到2013年左右，在经历了工作室的一些内部动荡后，id决定将整个项目推翻重做。在随后团队主心骨与元老的约翰.卡马克离职后，一支由退伍海军陆战队员、电影原画设计师和雷神之锤开发成员组成的混编团队接手了项目。

随后收购了id的母公司贝塞斯达宣布：预购《德军总部：新秩序》的玩家将获得参与DOOM新作的Beta测试资格。尽管由于Kotaku来源不明的开发困境泄露新闻，id面临着多方的质疑，但终究还是在2014年QuakeCon现场现观众展示了新作的一系列信息。过去的游戏标题被修改，简化成了带有软重启性质的《DOOM》。后来为了区分元祖作品，多数人将其称为《DOOM2016》。

游戏最被业界所知晓的一次展出，是2015年的E3贝塞斯达发布会。首先是贝塞斯达市场部门主管皮特.海因斯上台对id进行介绍，随后由本作的团队负责人马蒂.斯特拉顿向观众讲解游戏的特色。其中提到了他对DOOM核心理念的看法——BadAss恶魔、大Fu…真特么大的枪以及快准狠的游戏节奏。

随后是展会公布了一段实际演示。其快速而残暴的战斗场面令不少人感到惊叹，甚至一度出现了对其内容真实性产生质疑的声音。但最终在发售后的表现除了诸如主角头盔上的“F90”被换成了符文标志，以及一些敌人造型细节的修改外，整个游戏的表现与宣传片内容相比没有丝毫缩水。甚至实际游戏表现要比演示的还更上一层楼。

玩家再次扮演一位无名的太空海军陆战队员。但这一次在游戏的资料中，可以得到其名为[Doom Slayer]的绰号。其背景来历在一定程度上采用了模糊化的描绘方式，虽然有大量细节和暗示在对旧系列主角进行呼应，但都故意被陈述得模棱两可。这种做法虽然让不少新玩家产生了疑惑，但却大大刺激了社区讨论的活跃性。它抛弃了《毁灭战士3》缓慢、黑暗且注重惊悚内容的架构，虽然在视觉层面上有不少相似之处，但在游戏玩法和氛围上却明显是对元祖系列的重现。

作为一款延续了古典FPS精神的作品，《DOOM2016》虽然还保留着过去的迷宫探索元素，但在一定程度上弱化了这个环节。它着重强调的是快节奏的残酷战斗场面。有别于同期市面上绝大多数的FPS，《DOOM2016》更像是个第一人称视角下、以枪械作为主要攻击手段的动作游戏。在发售后不久，被许多玩家赋予了“枪中忍龙”的别名。

虽然由于外包给其它工作室负责的多人对战模式质量平庸脱了后腿，但《DOOM2016》仍然获得了媒体与玩家的好评。整体来说，它在单机战役方面的水准堪称近十年来最好的FPS游戏之一。在不少人已经厌倦了写实系和电影化脚本泛滥的市场下，本作以饱满、完善且可玩性绝佳的单人游戏模式征服了许多人。虽然有不少从未接触过这类设计理念的人感到不适应，但它对于一些更加喜欢探索、高强度厮杀的玩家来说着实诱人。而这种新旧时代审美观上的对立还不仅仅出现在玩家群体中，就连专业媒体也暴露了不适应老炮儿风格的情况。

IGN就是如此。也正因为给《DOOM2016》的评价显得十分不公平，导致IGN遭到了不少人的嘲讽。随之导致的滚雪球效应，甚至出现了每年都要拿这种随意打低分的行为来批斗IGN。

不过，虽然有诸如太暴力、路太难找、战斗没有掩体和上弹匣的设计莫名其妙等抱怨，但《DOOM2016》还是获得了肯定。随着时间的推移，除了战斗之外，它在叙事上的尝试、新颖的配乐模式都获得了一致好评。游戏在MC上的最终平均得分在87分以上，玩家的口碑也相当不错。不仅如此，许多业内人士也对本作赞叹有加。他们认为id做得最好的一件事，就是向业界和市场证明了游戏表达形式的多样性，在一个老派精神与现代风格之间找到了平衡点。值得所有的从业人员借鉴学习。

虽然很遗憾，游戏因为如今引擎等技术限制，不能提供过去那样开放的MOD创作条件，但id仍然准备了一个SnapMap模式。它的表现如果对比原始DooM自然是显得有些尴尬，但在如今的游戏里头也算是个值得花时间游玩的创作模式。何况它还能支持在家用机平台上使用，并上传到社区中分享，在便利性方面有着得天独厚的优势。

从数据上来看，《DOOM2016》的商业成绩在及格线以上。它在上市数周内完成了PC版50万份的成绩，在英国实体销量榜上仅次于索尼独占3A大作《神秘海域4：贼途末路》。虽然没有家用机版本的实际数据，但它在一年内完成了PC版本200万份的销量。对于计算机市场来说已经是个不错的表现。更何况，实质上《DOOM2016》在如今的行业标准里头，甚至都算不上是个一线大作的规模。因此在利润上，它的回报可能要优于不少3A级作品。

### **DOOM：永恒（DOOM：ETERNAL）**

| 开发团队：id Software                                      | 发售日期：2020年3月20日（PC，PlayStation 4，Xbox One，Stadia） |
| ---------------------------------------------------------- | ------------------------------------------------------------ |
| 创意总监：马蒂.斯特拉顿，雨果.马丁                         | 引擎 / 程序员：id Tech 7 / 比利.伊桑·汗，埃文.尤班克斯，约翰.卡马克（鸣谢） |
| 编剧：雨果.马丁，亚当.加斯科因，乔恩.莱恩，乍得.莫斯霍尔德 | 作曲 / 演奏：米克.戈登                                       |

2018年洛杉矶E3游戏大展上，发行商贝塞斯达在自家发布会上正式公开了系列第5部正统续作《DOOM：永恒》。在当时展出的只有一段概念宣传片，以CG动画的形式演绎了被地狱入侵的人类世界，最后以毁灭战士出场并与群魔交战的场面落幕。随后两位项目主要负责人雨果.马丁与马蒂.斯特拉顿上台做了简单的演讲，并表示正式的实际演示会在同年8月份的QuakeCon 2018正式亮相——这也是首次在QuakeCon上全球公开转播，而不是到场玩家专享的形式公布系列作品。

时间来到8月份，id如期向当初承诺的一样展示了三段实际游玩演示。雨果表示本作将会采用最新的id Tech 7引擎开发，它的特色是回归当初id追求的运行效率，能够在主机平台上实现1080p下60帧的速度。并且如果你有一个配得上的硬件，那么个人计算机就能以最高1000帧数的效率运行——而截至目前的硬件水准也已经证明了游戏可以达到将近500帧的刷新率。

因为这个引擎的特性，所以本作加入了实时恶魔破坏系统。在面对玩家的攻击时，敌人身上的血肉会随之找到破坏，呈现出各种触目惊心的杀伤效果。得益于引擎功能，它们完全不会对运行产生负担。

id对本作的目标之一是打造一个“DOOM宇宙”。它将会更大、更全面地展现这个系列的背景与世界观。游戏提供了将近30种恶魔，乃系列之最。另外在美术设计方面，本作很明显还原了初代的氛围，采用了一种介于漫画与黑暗幻想之间的感觉，相比前作严肃质朴的科幻风格显得更为“卡通”一些。当然，系列标志性的黑色幽默丝毫没有半点削弱就是了。

游戏原定于2019年11月19号发售，但随后id宣布将会延期以便对游戏做更多雕琢。由于这个决定导致它与任天堂旗下的《集合吧！动物森友会》十分巧合地在同一天发售，所以两部作品的粉丝进行了大量的互动，成为了2020年乃至游戏史上最大规模的玩家群体友好交流。这个事件一度被传为佳话，并诞生了无数的社区民间创作。

到了2020年游戏终于得以发售。玩家再次扮演扮演DoomGuy，面对全面入侵地球的恶魔军团，展开了一场的单枪匹马的复仇之战。游戏从前作的框架进行了更加精细、强调策略与操作能力的改良。它有着与如今市面上绝大多数同类第一人称射击游戏截然不同的体验，更加接近诸如《忍者龙剑传》、《鬼泣》或是《只狼》等高难度动作游戏的体验。

最典型的特色是鼓励玩家前进而不是寻求掩护的战斗思路。游戏中玩家可携带的资源量相当少，但每一场战斗中都能够通过击杀敌人来获取一切所需的补给。玩家装备了可以自动填装的肩部榴弹发射器，在一开始就拥有作为弹药补给机制的电锯；之后还能习得强力AOE攻击“血拳”和裁决剑等等，让毁灭战士更加如虎添翼。但代价就是普通的近身格斗被削弱为只能用于击退小兵的防御技能。

本作的恶魔被分为几个阶级——炮灰、重型、超重型以及BOSS级和环境型。这些恶魔总计约30种，每一个敌人在设计思路与应对方式上都截然不同，当它们在后期形成了配合后，呈现出的变化远远超越了截止目前任何一个已知单人FPS的认知范围。

id Tech 7引擎的恶魔破坏系统不仅仅是噱头。它们在被玩家击破身体部位时，也会随之出现战斗力上的骤降。比如肥球的火炮在损坏后其威胁也会随之下降，而机械蜘蛛头顶的炮塔也能够在交火时被拆下，减少它们的远程打击能力。

毁灭战士获得了新的冲刺技能以及强力的爪钩，所以在普通移动时被故意做得比过去稍微慢一些，以此来鼓励玩家多使用各种手段增加动能。这些移动机制也因此改变了关卡设计模式。在战斗之外，游戏还有大量的平台跳跃，毁灭战士需要在墙壁间攀爬，利用横杆摆荡或前往各种险要的地点。

以现代游戏的标准来衡量，《DOOM：永恒》的难度极高——甚至可以被列为2020年乃至本世代最难的游戏之一。它的设计理念同时兼具了战术思考与操作反应，对玩家自身的要求与学习能力都提出了相当严苛的标准，不是任何人都能轻易尝试的。即便是最低难度，它也远远超过大多数FPS，因为这点，许多游戏水平较差的媒体人和轻度玩家都无法适应，最后引发了一连串的社区讨论。

除了精彩之极的单人模式外，游戏还有一个id自行开发的[战斗模式（Battlemode）]。它采用了固定2对1的非对称玩法——两名玩家扮演恶魔，1人操作毁灭战士，在各种错综复杂但相对狭小的斗技场展开厮杀。恶魔玩家可以选择多种职业，它们单打独斗几乎无法在毁灭战士手中讨到任何便宜，但却可以召唤手下并利用各种计谋与陷阱来围剿这个强大的对手。而毁灭战士则需要在与单人模式相似的规则下，一边获取补给一边猎杀两名恶魔玩家。

游戏虽然没有微交易和内购，但同时也有如今游戏常见的季票。它将提供两个剧情DLC内容，但截止本书完成之前尚未发售，因此没有收录在其中。



**评价与发行后的争议**

由于COVID-19疫情在游戏发售时刚好处于爆发期间，因此本作的实体销量受到了一定程度影响，比前作低了约33%。但即便如此，它也比《DOOM2016》的销售速度快了将近3倍，在发售当月就完成了数字版300万份的销量。之后也多次成为Steam销售榜的冠军。

它被誉为有史以来最好的第一人称动作射击游戏之一，完成了系列的自我突破，对整个FPS类型来说也是极富历史价值的重要篇章。游戏获得了大量好评，在综合评分网站上拥有90分左右的平均分，但因为涌入了大量新玩家无法适应其高难度，所以用户评分相比前作有所下滑。但在系列老玩家的评价里，《DOOM：永恒》几乎获得了一片倒的赞誉。

不过，游戏因为发行商贝塞斯达强行推广自家平台——Bethesdanet，即便是游玩单人战役模式也仿佛要求检查账号，因此许多人对这种破坏流畅体验的做法感到不满。之后人们很快发现，由于Bethesdanet服务器一直存在的诸多Bug，甚至间接会导致游戏自身的存档出现问题。这些现象都让本作优秀的游戏素质蒙上了一层阴影。
