------

《毁灭战士3》（DOOM3，或风格化的写法DOOM³）仍旧由id Software继续担任开发工作。本作于2004年8月3日在北美Windows平台最先发售，而Linux与XBOX版则分别于2004年10月1日、2005年4月3日先后上市。由于机能限制，XBOX不仅在画面表现上略逊于高配置的PC版，同时还删除了绝大多数火星基地户外场景。

![](resource:assets/images/section_4/c2_01.jpg)

本作的亮点在于采用了当时最新的id Tech 4图像引擎，是该系列第一次真正采用了全3D图像技术的作品；这个基于OpenGL诞生的引擎后来也被用于诸如《雷神之锤4（Quake4）》或旧版《掠食（Prey）》等作品中。该引擎的主要特点在于光照技术的提升，能够实现投射到物体上的光影效果的即时运算效果；此外，本作也是id公司首次在自己的游戏里广泛使用了凹凸材质贴图。

虽然不及元祖系列那般成为了有史以来移植作品最多的游戏，但本作也算是一直有各种移植于再发售的版本。除了本篇游戏之外，本作还推出了一部资料片《邪恶复苏》、一个强化版《毁灭战士3：BFG》，并且还在手机平台利用原版素材推出了一个外传性质的《毁灭战士：复活》。

2011年11月22日，约翰.卡马克在删除了一部分可能涉及侵权的内容后，将id Tech 4引擎源代码免费公布于网络上以分享自己的成果。

### **毁灭战士3**

| 英文原名   | DOOM 3                         |
| ---------- | ------------------------------ |
| 发售日     | 2004年8月3日                   |
| 创意总监   | 蒂姆.威利兹                    |
| 设计师     | 蒂姆.威利兹 /  马尔.布莱克威尔 |
| 编程       | 约翰.卡马克 /  罗伯特.达菲     |
| 作曲       | 克林特.沃尔什                  |
| 编剧       | 马修.科斯特洛                  |
| 开发团队   | id Software                    |
| 代理发行商 | Activision                     |
| 登录平台   | Windows /  Linux / OS X / XBOX |

自1998年左右，大量有关Doom第三部正统续作的猜想与分析就广为流传。在最著名的Doom社区论坛[Doomworld]上，当时许多社区用户和玩家将其称为“DooM2000”，表达了希望能在进入新世纪同时便能看到新作来临的一种期盼。

然而与他们想象的有些不太一样，2000左右只是id正式宣布对系列最新作进行立项开发的年份。

2000年6月，约翰.卡马克在公司内部发布了一份关于对Doom系列进行重启的计划书。其中表示虽然遭到了艾德里安与凯文.克劳德等元老的反对，但公司内部多数成员仍然同意对系列最新作进行故事重启。

由于现实中火卫1其实非常小，因此这一次将舞台搬到了过去没有出场过的火星主基地上面。玩家再次扮演无名陆战队员，从入侵源头开始经历整个惨剧的全部过程。游戏着重了故事演出方面的表现，在场景复杂程度与细节上比以往更加丰富；除了一如既往痛宰恶魔大军外，编剧还为游戏加入了火星原住民等新的故事元素，整个世界观的格局比以往更加合理可信了。

由于在实际游戏内容中，玩家发现了大量隐藏着对前两部作品剧情的联系，让《毁灭战士3》看起来并非如宣传所述是完全重启那么简单。然而尽管如此，那些提出怀疑观点的人也拿不出更有力的证据，官方自己多年来也从未有过进一步的明确，因此本作通常在大多数人眼里仍旧被视为以新技术对初代故事重新诠释的重启作品。



**与实际游戏有所差异的Demo**

游戏在2002年的洛杉矶E3大展上正式亮相。id公开了一段15分钟的实际游戏影像，其中展示了新引擎技术下的画面效果、与过往截然不同的美术风格、以及游戏主要玩法和一部份系统机制等等。

![](resource:assets/images/section_4/c2_03.jpg)

在这段Demo宣传片里以恶魔入侵为开端，详细呈现了地狱亡灵腐蚀人类士兵并将其转化为怪物的过程。之后，一位无名的陆战队员登场，在展示了各种战斗与探索机制后，最终整段影像以玩家在面对地狱骑士时战败而亡结束。

虽然它的演示以彼时的水准来看很出色，但后来人们发现这段Demo与实际游戏有多处不同：

- 玩家角色在宣传片里会戴上头盔；但实际上本作主角是系列唯一露出面容的。
- 整个流程中的节奏要比正式版更加缓慢，人物移动倾向于写实系FPS。
- 僵尸无法被彻底击杀，倒地一段时间后还会再次爬起来攻击玩家。
- 透露了原本可能会出现潜行元素，玩家能够通过躲避强敌来增加生存机会。
- Demo里的霰弹枪在材质与造型上都更加精细。
- 被恶魔转化的突击队士兵，会说出各种诸如“死吧凡人”的中二台词。
- 更加丰富的演出脚本，在玩家死亡时还会出现被恶魔杀死的动画。

这段宣传片让Doom3在当年的E3展会上获得了包括[最佳展出游戏]、[最佳动作游戏]和[最佳PC游戏]在内的多个奖项，在当时获得了非常好的反响备受市场期待。然而，由于卡马克对于作品精益求精的原则，原本预计在2003年发售的计划最终被推迟到了2004年下半年才得以上市。



**风格的大幅转变**

为了突出新引擎的技术革新，《毁灭战士3》在游戏的风格上做了调整，更加强调渲染恐怖氛围并弱化了过去赖以成名的“跑射”节奏。本质上它仍旧是一个在封闭狭小的昏暗空间里，与各种角落里埋伏的恶魔作战并探索未知领域的游戏——只是变得更加缓慢、写实和惊悚。

虽然卡马克曾经为了反对同僚的提案，而说出了“游戏剧情不重要”这样的言论，但他还是为《毁灭战士3》聘请了一位专业作家——曾经为经典点击式冒险解谜游戏《第七位访客》撰写过剧本的马修科.斯特洛（Matthew Costello）。

游戏有着比过去更加多样的脚本演出，也穿插了不少过场动画来展现剧情。更重要的是，本作加入了类似《网络奇兵》等沉浸式模拟游戏的一大特色：文档、录音以及环境叙事。以此来增加整个故事的深度与解读乐趣，现在你不仅可以通过记载着无数值班记录、医学报道和日常闲聊的PDA来收集各种密码，还能了解到UAC员工的工作与生活情况。许多关于恶魔的研究记录也能够帮助你更加了解自己的敌人，甚至还有不少对于地狱和火星的文献资料以扩展世界观格局。

而这种利用文档记录游戏故事的方式，也能够让完全不想考虑剧情细节、专注于战斗与探索的玩家有所选择——你可以热衷于收集散落在关卡中的PDA来了解故事内幕，但即便完全不理会也没有关系。

此外，为了进一步体现人物与环境的互动性，Id还特意加入了一个计算机互动的系统——游戏拥有上百个可以进行触屏互动的显示屏幕，玩家可以通过指针的定位来实现对诸如开关、游戏机或密码锁的操作。这个点子其实早在1992年，由汤姆.霍尔编写的《DOOM圣经》里就已经有所提及了。



**游戏模式**

和以前一样，《毁灭战士3》除了主要的单人战役外仍旧备有多人在线游戏，提供了允许最多4名玩家同时进行的死斗模式。总共有四个不同的规则：标准的集体混战，争夺物资并消灭除自己以外所有玩家以获得最高分；规则与前者相似的团队死斗；以消耗对手生命数量为目标，战斗到只剩最后一人的类吃鸡死斗；以及最后的决斗模式。

由于默认能够参与的人数较少，所以在之后玩家自己开发了能够进行8到16人的MOD。此外随着Xbox Live已经全面停止服务，所以XboX版本的Doom3多人模式已无法再使用。

本作提供了四个难度选项，每个层级下敌人的伤害会逐步提高，但和系列所有作品的规则一样，不管任何难度下敌人的生命值与防御力都不会得到提升。最初玩家只能宣传前3个难度，在通关后可以选择噩梦——在这个难度下玩家最初便能拥有灵魂立方体，但是生命力会不断下降直到维持在25点，并且没有任何可拾取的医疗包补给。



**游戏在开发过程中的删减内容：**

- 大量的火星户外冒险关卡由于引擎限制最终被放弃。

- 一把功能类似远距离狙击枪的轨道能量武器被删除。
- 原本有一把能够进行火焰喷射的武器也被取消。
- PAD显示地图界面的功能最终没有实现。
- 女性NPC戏份被删除，其中包括一具裸露的女性尸体。
- 约有包括机械蜘蛛（Arachnotron）在内的4～5种恶魔仅停留在设定草案。



整个项目从立项到正式发售共花费了4年时间，耗资1500万美元。尽管绝大多数旧系列玩家和媒体认为本作在风格的转变上并不算成功，但游戏整体的出色表现仍旧取得了相当好的商业成绩，在上市后的一次统计中表示共售出了350万份以上。除外，舆论对其评价也是普遍维持在较高的水准线上，在综合评分网站Metacritic上维持着媒体87、用户平均7.6分的成绩。

### **毁灭战士3：邪恶复苏**

| 英文原名   | DOOM3: Resurrection of Evil                                  |
| ---------- | ------------------------------------------------------------ |
| 发售日     | 2005年4月3日                                                 |
| 创意总监   | 蒂姆.威利兹                                                  |
| 设计师     | 亚当.贝勒菲伊尔  /  帕特里克.胡克 / 布兰登.詹姆斯 / 戴维.开尔文 |
| 编程       | 布莱恩.马特 / 达林.麦克尼尔 / 肖恩.米切尔                    |
| 音效制作   | 克里斯蒂安.安特科                                            |
| 技术总监   | 约翰.卡马克                                                  |
| 开发团队   | Nerve Software                                               |
| 代理发行商 | Activision                                                   |
| 登录平台   | Windows /  Linux / XBOX                                      |

在《毁灭战士3》发售两个月后的2004年10月，id公布了扩展资料片《邪恶复苏》，作为游戏本篇后续剧情的延伸内容。游戏需要在原版基础上安装才可进行游玩，由id提供技术支援，Nerve Software负责主要的开发工作，而Activision公司仍旧是本作的代理发行商。

《邪恶复苏》于2005年4月3日在PC平台正式发行，随后于同年的10月份登陆微软XBOX主机，后者可以独立于原版的《毁灭战士3》即可运行，并且收入了《毁灭战士》、《毁灭战士2：人间地狱》及其扩展包《Master Levels》。XBOX版本为了让游戏体验更加轻松，也加入了显示敌人HP值的界面选项。

游戏包含了12个单人战役关卡，包括BOSS在内多种新敌人和两种新武器。多人模式从原本的4人死斗变成了允许最多8位玩家同时参与，并且加入了4张新地图。



**玩法上的改变**

本作在玩法上基本与原版的单人游戏模式一样。玩家扮演的是另一位陆战队员，在《毁灭战士3》结局的两年后重返火星遗迹探索时发现了古老的神器，导致地狱之门又一次被打开，玩家需要击败三个地狱猎人吸收它们的力量来强化神器，最后一劳永逸地击败在前作中转化为恶魔的彼苏格博士。

本作加入了在《毁灭战士3》开发阶段就设计完成却因为设计理念不符而被取消的[Grabber Gun]。玩家可以利用这把武器搬运和发射各种小型物体、获取远距离的道具、吸收敌人发射的能量弹并反击回去、甚至直接吸死迷失亡魂这样的小型敌人。由于使用原理类似于《半条命2》著名的“重力枪”，所以在游戏发行后被一部份舆论质疑抄袭，不过在事后开发人员表示，该武器原本就是用于《毁灭战士3》内部测试版本时用于移动错误物品的开发者工具，而后者在开发和上市时间上都早于《半条命2》，并不存在抄袭一说。

另一把新加入的武器是双管霰弹枪，这把经典武器的回归使得《毁灭战士3》拥有了系列传统的通用弹药机制，更加强力的杀伤力弥补了原版常规实弹系武器偏弱的问题，让玩家可以利用它快速击杀敌人。

最后一个新增武器是代理灵魂立方体的[The Artifact]，这是一个悬浮在玩家人物掌中的心脏型器物，能够在发动后实现强化角色的能力。玩家每次击败3个地狱猎人BOSS后，会逐步解锁子弹时间、攻击力增强和无敌三种效果。你可以利用场景中的人类尸体对神器进行充能，总共可以储存3次使用机会，在发动后便会获得全部的增强效果。

游戏的剧情核心也是围绕着神器展开。讲述了主角不断寻找线索挖掘神器秘密，一路击败追杀的恶魔，最后在前往地狱将其摧毁。在游戏最终结局里，陆战队员将神器塞入Maledict口中将其击杀，而神器也在这个过程中被销毁。

虽然在原始版本发售时就饱经抨击，但是这款资料片仍旧没有改进玩家照明手段的问题。在XBOX版本里，玩家的初始手枪上挂载了手电筒，一定程度上改善了在探索与战斗之间总是需要来回切换武器的问题，但由于更加强力的武器并没有获得这项功能，所以后期仍旧有流程拖沓的现象。



**新增加与改变的敌人**

除了4位BOSS级敌人外，本作还增加了两种全新的敌人——Vulgar与Bruiser。

前者是一种类似Imp的中等体型恶魔，它的建模实际上来自于原版《毁灭战士3》一个被取消的敌人设计草案。这种同样擅于在通风管和狭窄地形攀爬的怪物替代Imp成为了本作的一线杂兵，投掷蓝色的镭射球并且拥有更高的HP，玩家可以利用Grabber Gun与其进行“投球运动”来轻松的杀死它们。

Bruiser则是一种极为强大的精英恶魔，它们甚至比地狱骑士还要难缠。这种双手都安装了手炮的敌人会一边前进一边火力压制对手，强大的生命力与攻击频率使得玩家不得不借用神器的力量才能快速将其击败。

除此之外，原版的僵尸加入了一种穿着化学防护服的版本，而迷失亡魂的造型也一改《毁灭战士3》的飞行人头，回归了更加野性的骷髅形象，与旧系列风格相符。

《邪恶复苏》获得了与原版相近的评价，虽然有一定的质疑声认为抄袭了《半条命2》的灵感，但在了解到实际情况后都对新的武器给予了正面评价。在绝大多数的综合评分网站上，本作的平均分一般都在80以上。此外，由于德国法律法案的规定，本作与系列乃至id的许多作品一样，都被禁止在该国家进行宣传推广和贩卖。

### **毁灭战士：复活**

| 英文原名   | Doom Resurrection                       |
| ---------- | --------------------------------------- |
| 发售日     | 2009年6月26日                           |
| 创意总监   | 约翰.卡马克 /  汤姆.穆斯塔因            |
| 制作人     | 凯瑟琳.安娜.康                          |
| 编程       | 约翰.福肯伯里 / 肖恩.格林 / 乔什.马特尔 |
| 开发团队   | Escalation Studios / id Software        |
| 代理发行商 | id Software                             |
| 登录平台   | IOS                                     |

随着移动平台的兴起，一向对尖端技术感兴趣的卡马克也想试试在手机上开发一些id旗下的经典作品，于是我们便看到了《毁灭战士 RPG》和《狼穴RPG》等作品的出现。而在2009年《毁灭战士3》发行的4年后，id又将其素材再利用，制作了《毁灭战士：复活》这款外传作品。本作除了由卡马克担任团队主创外，他的妻子凯瑟琳安娜康也以游戏制作人的身份参与了进来。

故事剧情讲述了在《毁灭战士3》中的B小队一名幸存者在醒来后，遇到了加雷特博士和一个名叫Sam的智能机器人，并在它的指引下与恶魔战斗，解救UAC幸存人员并最终帮助幸存者们搭上飞船逃生。

虽然看起来游戏是与本篇故事时间线并行，然而游戏的时间线却是发生在2145年6月——《毁灭战士3》故事开始时的几个月前，所以本作事实上只是另一个平行宇宙中的故事，之后的一些细节也显示了两者不在同一个世界线里。

游戏总共有8个关卡，基本上都是正篇游戏中出现的场景，而出于机能和玩法上的限制，同屏最多只能显示2～3名敌人.玩家一开始便拥有突击步枪，并且储弹量也是无限的，只有类似霰弹枪和BFG这样的特殊武器才有着弹药限制。

虽然在视觉风格和素材上延续了《毁灭战士3》，但实际上画面进行了大幅度缩水，游戏方式也变成了类似街机上《死亡之屋》那样的轨道射击玩法。游戏运用了手机的感应功能，当敌人投掷火球时，可以通过摇晃手机屏幕来侧身闪避攻击。而一旦被丧尸抓住，玩家也需要快速摇动手机来挣脱。

本作于iPhone的iOS系统上发售，在国内可能很少有人接触过。虽然整体上游戏的玩法和节奏很适合在手机平台上游玩，不过作为一个10年前的实验性作品来说，和现在已经在某些方面极为成熟的便携游戏相比，并没有什么值得一试的内容。仅仅只是系列爱好者们的猎奇产物而已。

有趣的是，在本作开发阶段，向来很少提及过去的卡马克，居然在当年的采访时坦言了《毁灭战士3》的手电筒设计是该游戏犯的最严重错误，光是这点也许就让本作有了一些谈资了吧（笑）。另外还有一个值得提及的是，在本作中最后为了帮助主角而牺牲的Sam，在后来的《毁灭战士II RPG》中被修复了，简介验证了本作是被划分在了“RPG系列”的世界线里。

### **毁灭战士3：BFG版**

| 英文原名   | DOOM 3：BFG                                                  |
| ---------- | :----------------------------------------------------------- |
| 发售日     | 2012年10月16日                                               |
| 执行制片人 | 埃里克.韦伯                                                  |
| 设计师     | 蒂姆.威利兹                                                  |
| 编程       | 柯蒂斯.阿林克 / 罗伯特.达菲 /  杰弗里.法朗 / 瑞安.格列夫 /  <br />比利.伊桑.汗 / 格洛丽亚.肯尼克尔 / 迈克尔.梅纳德 / 约翰.罗伯茨 / 史蒂芬.塞拉芬 / 扬.保罗.范.韦弗伦 |

| 登录平台   | Windows / PS3 / XBOX360                                      |

2016年10月16日作为画面重制与内容合集的《毁灭战士3：BFG》在北美发售，登陆了PC、PS3和XBOX360平台。本作由id进行开发，母公司Bethesda发行。在登陆该世代主机平台后，游戏的画面技术得到了提升，加入了一些新的功能修改以及检查点自动存档机制，支持3D眼镜技术并增加了新的游戏内容。



**该版本共收录了**

- 《毁灭战士3》
- 《毁灭战士3：邪恶复苏》
- 《终极毁灭战士》
- 《毁灭战士2：人间地狱》
- 原本仅在XBLA上提供下载的2代扩展资料片《没有休息日的生活》
- 基于原版扩展的新增单人战役模式《失落的任务》



**与原版相比在性能上的改变**

- 在PC平台上运行的所有游戏都可以兼容Windows 7操作系统。
- 主机平台上的画面效果与高配置下的PC一致。
- 增强了音质质量。
- 删除了高级键盘映射功能。
- 支持3D电视和高分辨率显示器。
- 改善了游戏运行效率
- 游戏可以在120hz下运行，拥有更加流畅的画面体验
- 一定程度上改进了贴图质量
- 光源效果获得了更好的提升，来自手电等光照都可以投射阴影
- FOV最大值被下调了



**游戏机制上的改变**

- 增加了便携式照明系统，玩家终于可以在不切合武器的前提下兼具战斗与探索
- 原本作为武器的手电筒被完全删除
- 增加了更多的补给，玩家的盔甲也能达到125点
- 拥有检查点自动保存机制，通常为每关进行一次自动存档
- 取消了视野缩放功能
- 在关卡中增加了更多的敌人
- 《邪恶复苏》的关卡再也不需要佩戴防毒面罩，主角可以直接进入充满了有毒废料的排污系统中。



**失落的任务**

玩家将在这个新增战役模式里扮演《毁灭战士3》本篇剧情里失踪的B小队唯一幸存者，在你从恶魔的袭击中死里逃生后，一位自称在UAC远程传送实验室的幸存者理查德迈耶斯博士（Dr.Richard Meyers）联系上了玩家，希望你能帮助他摧毁被恶魔们抢夺的传送装置。

由于这个装置上面的信号坐标能够让大量恶魔直接前往地球，而恶魔们已经将其带入地狱，因此必须有人进入地狱去摧毁这个装置。玩家最后成功的毁掉了传送装置，并且和理查德博士一起被地球的增援部队所救，回到了地球。

该附加任务共有7关，基本上是对原版内容的一次吃书，尤其是《毁灭战士3》提及的幸存者人数在这段剧情里被扭曲。不过在设计上，《失落的任务》仍然得到了一定程度的好评，认为对原版内容的浓缩有相当的游玩价值。

《毁灭战士3：BFG》的评价褒贬不一，它在兼容性和音画效果上有了不错的提升，合集包式的内容也十分厚道，具有收藏意义。但是游戏的一些小问题，诸如在主机上的加载速度和强制存档等等效果令人失望，对游戏体验造成了一定的影响。最令人尴尬的是虽然解决了照明问题，允许玩家在昏暗地点不用切换武器，然而玩家却又对手电筒被删除感到了不满。
